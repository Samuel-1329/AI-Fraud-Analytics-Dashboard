# 💳 AI-Powered Fraud Detection & Financial Analytics Dashboard

An interactive Machine Learning and Financial Analytics dashboard built using Python, Streamlit, Random Forest, and Plotly to analyze and detect fraudulent financial transactions.



# 📌 Project Overview

Financial fraud is one of the biggest challenges faced by banks and digital payment systems. This project uses Machine Learning and Data Analytics techniques to identify fraudulent transactions and visualize fraud patterns through an interactive analytics dashboard.

The dashboard provides:

* Fraud analytics and visualization
* Machine Learning model evaluation
* Fraud trend analysis
* Business insights and recommendations
* Interactive charts and KPI metrics

This project demonstrates practical applications of:

* Data Analytics
* Machine Learning
* Financial Intelligence
* Business Analytics
* Dashboard Development



# 🚀 Features

## 📊 Interactive Analytics Dashboard

* Fraud transaction monitoring
* KPI metrics and fraud statistics
* Interactive Plotly visualizations

## 🤖 Machine Learning Integration

* Random Forest fraud detection model
* SMOTE balancing for imbalanced data
* Fraud classification analysis

## 📈 Financial Analytics

* Fraud trend analysis over time
* Transaction amount analysis
* Fraud percentage insights

## 🔥 Model Performance Evaluation

* Confusion matrix visualization
* Feature importance analysis
* Correlation heatmap

## 🧠 Business Intelligence

* Executive summary
* Fraud detection insights
* Banking recommendations
